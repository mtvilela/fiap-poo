O código permite interagir com o objeto Projetor através das seguintes ações:
Ligar/Desligar: Altera o estado do projetor (com verificações para evitar comandos repetidos).
Controle de Volume: Aumenta ou diminui o volume em intervalos de 5 unidades (apenas se o aparelho estiver ligado).
Exibição de Status: Mostra na tela se o projetor está ativo e qual o nível atual do volume.
